# AirtestIDE 启动方式

## Windows用户

- 下载官网的zip包，解压后得到AirtestIDE文件夹
- 双击AirtestIDE/AirtestIDE.exe即可启动
- 在部分机器上可能由于显卡兼容性问题，导致无法正确启动，或脚本编辑窗口无法编辑，可以双击目录下的`兼容模式启动.bat`文件来启动

## 遇到问题？

我们的官网文档提供了教程、常见问题、使用指南等信息，欢迎访问：http://airtest.netease.com/docs/cn/index.html

- 新手上路系列教程：https://airtest.doc.io.netease.com/
- 手机连接遇到问题：http://airtest.netease.com/docs/cn/2_device_connection/2_android_faq.html
- 问题与BUG反馈，请在这个页面 https://github.com/AirtestProject/AirtestIDE/issues 创建issue，并附上详细的问题描述、报错信息、相关截图等，方便我们定位问题。


# AirtestIDE Help

## Windows users

- Download the official website zip package, extract the AirtestIDE folder
- Double click on AirtestIDE/AirtestIDE.exe to launch
- On some machines, it may not be able to start properly due to graphics card compatibility issues, or the script editing window cannot be edited. You can start by double-clicking `CompatibilityMode.bat` file in the directory.

## Encounter problems?

Our documentation provides tutorials, frequently asked questions, user guides, and more. Please visit: http://airtest.netease.com/docs/en/index.html

- Beginner's series tutorial: https://airtest.doc.io.netease.com/
- If you're having trouble connecting to your phone: http://airtest.netease.com/docs/en/3_faq/android_connection_new.html
- Questions and bug feedback, please create an issue on this page https://github.com/AirtestProject/AirtestIDE/issues with detailed description of the problem, error messages, related screenshots, etc., so that we can locate the problem.
